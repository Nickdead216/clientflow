# ClientFlow

Mini CRM para gestión de clientes, proyectos y tareas.

## 🚀 Tech Stack
- Frontend: React + Vite + TypeScript
- Backend: NestJS + Prisma
- Database: PostgreSQL

## ✨ Features
- Registro y login con JWT
- CRUD de clientes
- CRUD de proyectos
- CRUD de tareas
- Autenticación protegida

## 📦 Instalación

### Backend
```bash
cd backend
npm install
npm run start:dev
